Conversor de Moneda

Desarrollado en Java, este proyecto consiste en una aplicación que permite convertir diferentes monedas utilizando tasas de cambio en tiempo real. Implementé una API para obtener los datos de cambio de divisas, asegurando precisión en las conversiones. Este proyecto me permitió profundizar en la programación orientada a objetos y en la integración de servicios web.

Encriptador de Texto

Utilizando JavaScript, HTML y CSS, creé una herramienta web que encripta y desencripta textos. La aplicación ofrece una interfaz amigable, permitiendo a los usuarios proteger su información. Este proyecto no solo me ayudó a mejorar mis habilidades en desarrollo frontend, sino también a entender mejor los algoritmos de encriptación y la seguridad de la información.

Juego del Número Secreto

Desarrollado inicialmente en JavaScript, el juego consiste en adivinar un número generado al azar. Luego, implementé la lógica del juego en una interfaz HTML y añadí estilos con CSS para mejorar la experiencia del usuario. Este proyecto fue una excelente oportunidad para combinar la lógica de programación con el diseño web, creando una aplicación divertida e interactiva.
